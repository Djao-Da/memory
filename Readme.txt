Memory is a game when you need to open all cards use minimum tries.
You can open only 2 cards, then if they not match (equal), the cards hide, if the cards match, they stay opened.

Program is written by directional style, I’d like to write it in OOP principles soon.

You can copy this code in to http://www.codeskulptor.org/
or use desktop idle just needs to install «simpleguics2pygame» module 
http://simpleguics2pygame.readthedocs.io/en/latest/#

enjoy!
